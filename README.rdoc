= poolparty-extensions

  Extensions to PoolParty!

  Just install and include in your clouds.rb

  clouds.rb
    require "poolparty"
    require "poolparty-extensions"

= Available extensions

== dynomite
  Installs edge dynomite from the git repository
  
  Usage:
    enable :dynomite

== tokyo_tyrant
  Install tokyo tyrant
  
  Usage:
      enable :tokyo_tyrant
